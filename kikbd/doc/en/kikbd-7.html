7. Bugs

  • Some times when you did Apply from configuration program with running kikbd, it locks up and you have to kill it.
  • Incorrectly hangling X11 events. From some windows it does not receive events
  • You can have problem with screen lockers, but with KDE screensavers everything must be Ok. 
         % kikbd -display <your locked display> -rotate
    This rotate keyboards in the usual kikbd way and you can unlock you display
Please report bugs at http://buglist.kde.org
