    • Introduction

    KEdit is a small, simple and almost featureless editor, just like the well-known Notepad. It is suitable only for small editing tasks or as a simple textfile browser. KEdit is intendet also to those many users who have to edit some resource-files not knowing how to use the stone-age editor vi. Since version 0.3.0 kedit is capable to handle URLs, meaning that you can read and write files via ftp similar to kfm.

    Installation

    Requirements: You have to set the environment variables QTDIR and KDEDIR first. Further you need the libraries

    • libkde 0.6.1
    • libkfm out of kfm 0.5.1

    installed. Then a simple make and make install should do.

    Using KEdit

    Since the functionality is strongly limited you will not have any difficulties in using kedit. For the transparent network access type for example

    kedit ftp://ftp.kde.org/pub/apps/editors/kedit-0.3.1.lsm

    to see how it works.
