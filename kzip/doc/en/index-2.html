2. Installation

2.1 How to obtain KZip

KZip can be found in the kdeutils package of the K Desktop Environment http://www.kde.org. ftp://ftp.kde.org/pub/kde/ is the main ftp site of the KDE project.

2.2 Requirements

  • The latest version of KDE. KDE is in beta so not everything will work perfectly. It can be found on ftp://ftp.kde.org/.
  • GNU Tar v1.12 and a recent gzip.
  • KFM is needed to get full functionality from KZip.
  • Qt v1.31. Information regarding Qt can found on http://www.troll.no/.
  • The appropriate programs for other archive formats are needed as well. These include lha, zip, unzip, and ar.
