2. Supported Platforms

  • RedHat Linux 5.0 : My development platform. Fully tested on it.
  • Other Linux, BSD : Should compile and run well.
  • Solaris : It won't work until somebody makes a replacement for snprintf. Of course, one should be able to install another libc to solve this...
  • AIX : I've heard about problems about localtime, ctime, and assorted functions.
I would be glad to receive comments about compiling and running ktalkd on other systems.
