2. KFM Configuration: Color

In this screen you can customize the File Manager color settings. When you click on a color box, the "Select Color"screen allows you to select a color for the corresponding item.

  • "Background Color" is the color of the background in a File Manager window.
  • "Normal Text Color is the color used for displaying normal clickable hypertext links for moving around within a hypertext document.
  • "URL Link Color" is the color in which clickable URL links (to local files, or Internet addresses) are shown, before the link is followed.
  • "Followed Link Color" is the color in which clickable links (both URL links, and Normal Text links) are shown, after the link has been followed.

Two further buttons control how links are displayed:

  • If "Change cursor over link" is selected, the cursor changes from an arrow to a pointing hand when it is over a clickable link.
  • If "Underline links" is selected, clickable links are underlined.
