1. Introduction

Activating ``enable system sounds'' allows sounds to be assigned to the various system events listed in the Events box. When such an event (like ``Window open'') occurs, the sound file that has been associated to that event will be played.

Currently, only WAV files are supported by the KDE System Sound Module. The default location of such WAV files is the directory $(KDEDIR)/share/sounds, and any WAV files available there will be listed in the Sounds box. You can also add other sound files to the Sounds list by, just dragging them from a kfm window onto the sound list.

To associate a sound to an event, use the pointer to highlight its entry in the Sounds list. (You may now hear it by clicking on the Test button.) Then highlight the corresponding event you wish to associate with it and click on Apply (or OK if you are finished).
