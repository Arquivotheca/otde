6. Advanced Window Manager Settings.

This screen allows you to configure various advanced Window Manager features.

In the Keyboard and Mouse settings:

  • If Ctrl-Tab walks through desktops is selected, this key combination can be used to switch desktops.
  • The Alt+Tab key combination is used to switch focus between different windows, making each window active in turn. If Alt-Tab is limited to current desktop is selected, only the open windows in the current desktop will be cycled through. Using the Alt-Tab mode list box, the action of the Alt-Tab key can be chosen to be in the graphical KDE style, or the "Classic" CDE style.
