2. The configuration modules

This is a list of the standard configuration modules provided by the KDE base packages. Please note that there might be many more on your system if you install additional software. You can access the help for all modules by first selecting the module in the control center and the pressing the help button.

  • Bell configuration
  • Memory inforation
  • Processor information
  • Keyboard configuration
  • Mouse configuration
  • Titlebar button configuration
  • Titlebar configuration
  • Window configuration
  • Language setting
  • System sound selection
  • Background configuration
  • Screen Saver configuration
  • Color Scheme configuration
  • Style configuration
