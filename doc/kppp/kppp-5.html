5. Using kppp

5.1 Command Line Options

kppp supports the following command line options: 

% wuebben: kppp -h
% kppp -- Valid options:
% -h describe command line options
% -c account_name : connect to account account_name
% -r rule_file: check syntax of rule_file
% -q quit after connection has been terminated

  • -h will describe the available command line options.
  • -c will try to connect straight away to the given account without bringing up kppp's main dialog. This option may be interesting to the impatient.
  • -r will check the specified rules file for valid syntax. This options is very useful when writing new phone time rules files.
