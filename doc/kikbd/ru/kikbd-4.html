4. Настройка

Для настройки используется программа kcmikbd, которую можно запустить из шела или из KDE Control Center.

4.1 Раскладки

Используя кнопки Добавить,Удалить, Вверх и Вниз можно отредактировать список раскладок, доступных при работе. Самая верхняя раскладка становиться основной при запуске KDE.

4.2 Переключатель и Алтернативный переключатель

Для переключения раскладок и набора алтернативных символов выберите удобные для вас комбинации. Если выбор Алтрнативного переключателя заблокирован, значит либо в вашем списке раскладок нет раскладки содержащей алтернативные символы или вы выбрали в качестве основного переключателя одиночную клавишу.

4.3 Сигнал

Вы можете слышать "Бип" всегда, когда меняется активная раскладка.

4.4 Специальная настройка

World Menu

You can have popup menu in any active window by pressing Switch keys for a long time (hold Right Control and Right Shift as on the picture and wait for popup menu).

Emulate CapsLock

For some keyboards (for example Russian) there is some problems with default Xserver CapsLock behavior. You can use KiKbdemulated CapsLock to fix this problem. With CapsLock emulated you can control Caps state by the color of the KiKbdbutton instead of the keyboard Led.

Input

You can select one of the:
  • Global - "standard" behavior, your keyboard applies to all windows.
  • Window - "extended" behavior, each toplevel window remember it's own keyboard
  • Class - "special" behavior", each window manager class remember it's own keyboard. For example, if you switch to Russian in one of the Kedit, you will get Russian in any other Kedit automatically.

Save Classes

If you select Input for Classes you can save the keyboard to WM Class association on the end of the each KDE Session and restore it at the beginning of the KDE Session. For example, if you leave Russian keyboard in the Kedit window today, you will get it automatically when you login tomorrow.

4.5 Style

You can customize KiKbd button Font and all of the four colors:
  • Foreground Color - Color of the Label Text
  • Background Color for normal state (No Alt Switch and no CapsLock)
  • Background Color when CapsLock emulation used and current Caps state is On.
  • Background Color while Alt Switch pressed.
