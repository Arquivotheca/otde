1. Introduction

KiKbd is designed to help typing national symbols different from default Latin symbols on Xserver.

It is different from XKB because it does not use XKB extension of Xserver.

It different from classic xmodmap when you load special xmodmap for you language and when use some special Mode Switch keys.

KiKbd use xmodmap method to setup national symbols for X11 applications, but perform dynamically switching of them. This means what KiKbd can support more then one language at the same time.

As additional of dynamic technique, KiKbd can do some nice job for your. For example it can setup different mapping in different windows or different applications (using Window or Class input style).
