5. Menu

The menu of kcc consists of only two entries: File and Help. The file menu contains the option Exit to close the Command Center.

The menu Help has several entries:

Help about KCC

Displays the help file for the Control Center (this file).

Help about modulename

Displays the current configuration modules help file, if a module is selected.

Using KCC...

Shows information about the Control Center.

Help for modulename

Shows information about the current configuration module, if a module is selected.
