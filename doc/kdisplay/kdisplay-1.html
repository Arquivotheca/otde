1. Introduction

KDisplay is a display setup utility. Running 

 kdisplay -setup
gives a tab dialog with five pages dealing with different aspects of the desktop display.

  1. The desktop Background.
  2. The Screen Saver.
  3. The desktop Color Scheme.
  4. General desktop preferences.
  5. Input device setup.

Running 

kdisplay -init
simply applies your preferences to the desktop.
