2. Installation

2.1 How to obtain KLJetTool

KLJetTool is part of the KDE project http://www.kde.org. KEdit can be found on ftp://ftp.kde.org/pub/kde/, the main ftp site of the KDE project.

2.2 Requirements

In order to successfully compile KLJetTool, you need the latest versions of libkdecore and libkfm. All required libraries as well as KLJetTool itself can be found on ftp://ftp.kde.org/pub/kde/.

2.3 Compilation and installation

In order to compile and install KLJetTool on your system, type the following in the base directory of the KEdit distribution: 

% ./configure
% make
% make install

Since KLJetTool uses autoconf you should have not trouble compiling it. Should you run into problems please report them to the KDE mailing lists.
